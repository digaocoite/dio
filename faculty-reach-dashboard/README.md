# AY25–26 Faculty Reach Dashboard

A standalone, Mizzou-themed interactive dashboard for exploring AY25–26 faculty reach by program.

## Features
- Searchable program list
- Facilitator filtering
- College-distribution mini bar charts
- Animated card reveal on scroll
- No dependencies (plain HTML/CSS/JS)

## Run
Open `index.html` directly in any modern browser.

## Notes about data accuracy
The data shown is the provided AY25–26 snapshot. Update the `programs` array inside `index.html` as new audited numbers arrive.
